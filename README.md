# pulse-feed

Published JSON feed for the **Pulse** news app (generated by the Pulse aggregator: `pulse digest`).
Static, public, served via GitHub Pages. Contains aggregated public-news metadata only (titles, links, summaries, source names) — no secrets, no user data.

- `feed/digest.json` — daily digest (top stories + per-category previews)
- `feed/categories.json` — category index
- `feed/category/<key>.json` — per-category feeds

Refreshed periodically from the Pulse backend on Amaterasu.
